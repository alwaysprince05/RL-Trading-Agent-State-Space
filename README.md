# 3D RL Trading Agent State Space Visualization

This project visualizes how a reinforcement learning (RL) trading agent explores and learns in a simulated financial environment. It animates the agent's state transitions (price, momentum, volatility) in 3D, coloring each point by the agent's action (buy, sell, hold) and showing the learning trajectory over time. A Q-value surface overlay illustrates the agent's evolving value function.

## What is this project about?
- Simulates an RL trading agent's exploration of the decision space
- Visualizes agent states in 3D (price, momentum, volatility)
- Colors points by action (buy/sell/hold)
- Animates the agent's learning process and trajectory
- Optionally overlays a Q-value surface

## Installation & Running
1. Clone the repository:
   ```bash
   git clone https://github.com/alwaysprince05/RL-Trading-Agent-State-Space.git
   ```
2. Install requirements (matplotlib, numpy):
   ```bash
   pip install matplotlib numpy
   ```
3. Run the script to view the visualization:
   ```bash
   python rl_trading_3d_visualization.py
   ```
